# /src

This folder contains the **Agent Orchestrator** demo code for the session.

## 📁 Contents

### 🐍 [`python/`](python/)

The Python implementation of the multi-agent property search system using:
- **FastAPI** - Modern async web framework
- **WebSockets** - Real-time agent communication
- **GitHub Copilot SDK** - AI agent runtime
- **SQLite** - Lightweight property database

See [python/README.md](python/README.md) for setup instructions and documentation.

## 🚀 Quick Start

```bash
cd python
pip install -r requirements.txt
python app.py
```

Then open http://localhost:8000 in your browser.
